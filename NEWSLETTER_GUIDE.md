# Greens Screens Ent — Newsletter Guide

This guide defines the standards for the "The Frequency" monthly newsletter.

## 1. Frequency & Distribution
- **Cadence**: Monthly.
- **Release Date**: The 15th of each month.
- **Data Collection Window**: From the 16th of the previous month to the 15th of the current month.
- **Year**: 2026 (Operational Era).
- **Core Value**: This newsletter is designed to bring **BREAKING NEWS** and **PREMIER UPDATES** to its readers. Avoid "old" news unless there is a significant current-month development.

## 2. Branding & Visuals
- **Primary Aesthetic**: Cyber-industrial, high-energy ("SIGNAL_ALWAYS_GREEN").
- **Core Palette**:
  - Background: Black/Dark Green (`#050a07`).
  - Brand Green: Neon Green (`#00ff88`).
  - Brand Lime: Neon Lime (`#aaff44`).
- **Section Colors (Contrast Rule)**:
  - **Major Release**: Neon Green (Primary).
  - **Indie Release**: Neon Lime.
  - **Tech**: Electric Cyan/Blue (`#00cfff`).
  - **Entertainment**: Amber/Gold (`#ffd700`).

## 3. Section Architecture
The newsletter MUST consist of exactly 4 topics in this order:

1. **Major Release (`releases`)**
   - Focus: **NEW LAUNCHES ONLY**. AAA games released exactly within the specified data collection window (e.g., April 16, 2026 - May 15, 2026). OLDER GAMES FROM PAST YEARS ARE STRICTLY FORBIDDEN.
   - Strict: No updates, DLC, early access patches, or rumors.
   - Count: 3 stories.
   - Links: Direct Steam Store links ONLY.

2. **Indie Release (`indie`)**
   - Focus: **NEW LAUNCHES ONLY**. Significant indie titles released exactly within the window. OLDER GAMES FROM PAST YEARS ARE STRICTLY FORBIDDEN.
   - Count: 3 stories.
   - Links: Direct Steam Store links ONLY.

3. **Tech (`tech`)**
   - Focus: Hardware, engine updates, studio tech announced or released in the current window.
   - Count: 2 stories.
   - Links: **PROHIBITED**.

4. **Entertainment (`entertainment`)**
   - Focus: Movie/TV adaptations, cultural media events from the current window.
   - Count: 2 stories.
   - Links: **PROHIBITED**.

## 4. Link Integrity Protocol
- **Game-Only Links**: Only `releases` and `indie` sections support links.
- **IDENTITY LOCK (STRICT)**: 
   - The ONLY allowed link type is `https://store.steampowered.com/app/[ID]/[NAME]`.
   - **Direct Profile Only**: The link must be the direct game profile link for the relevant game in focus per the article.
   - **Mismatched Links Forbidden**: You must NEVER link to a different game, an older version, a sequel, or another game in the same franchise. If you are talking about Game X, you cannot provide a link to Game Y.
   - **New Release Guard**: Only provide links for games that are physically out or have a live 1.0 store page.
   - **Sequel Guard**: Never link to an older game in a series.
   - **Match Protocol**: Link MUST correspond 1:1 with the installment discussed.
   - **Validation**: If the specific game does not have an active Steam store page, you MUST set `hasLink: false`. Discarding a link is better than providing an incorrect one.

## 5. Content Formatting
- **Blurb**: A 1-sentence "Hook" (max 15 words).
- **Full Summary (Research Digest)**: A 3-4 sentence deep-dive (80-100 words). It MUST contain technical data and unique insights.
- **Uniqueness**: The Summary must NOT repeat the Blurb.
- **Tone**: High-energy breaking news.

## 6. Visual Identity
- **Header Icons**: Replaced with a pulsing green orb (Neon Green/Lime).
- **Section Colors**:
  - Major Release: Neon Green.
  - Indie Release: Neon Lime.
  - Tech: Electric Cyan ($00cfff).
  - Entertainment: Amber Gold ($ffd700).
