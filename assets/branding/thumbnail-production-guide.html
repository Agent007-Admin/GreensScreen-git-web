00 · How to use this guide
THE WORKFLOW
Every time you create a new thumbnail, this guide is your first stop. It tells you which template to open, exactly which fields to update for your specific video, and which elements to leave completely untouched. The brand stays consistent. Only the content changes.
3-STEP PROCESS FOR EVERY THUMBNAIL
STEP 01
PICK THE TYPE
Ask: what kind of video is this? Game clip, review, or brand/general content? Your answer determines which HTML file to open.
STEP 02
CHANGE ONLY WHAT'S LISTED
Each type has a short list of fields to update — the title, game name, score, etc. Change only those. Everything else is locked brand elements.
STEP 03
RECORD OR EXPORT
Open the HTML file in a browser. Use OBS Browser Source to record it as an mp4, or screenshot it for a static thumbnail image.
01 · Decision
WHICH TYPE?
Three questions. One answer. Run through these every time — it takes ten seconds and keeps the system consistent.
WHAT TYPE IS THIS VIDEO?
Is this a highlight, clip, play, or gameplay moment?
→

Is this a dedicated review, deep-dive, or verdict on a specific game?
→
TYPE B — GAME REVIEW
Is this brand content, a community update, collab, or general video?
→
TYPE C — BRAND / GENERAL
Still unsure?
If the video is primarily about a specific game moment → Type A.
If the video renders a judgment or opinion on a game → Type B.
If neither of those is true, or the video is about GSE itself → Type C.
02 · Type A
TYPE A — GAME CLIPS
Use for: highlight reels, clutch plays, funny moments, montages, any video where the primary content is gameplay footage. The diagonal slash design communicates speed and energy. It fades to black at ~3 seconds and cuts directly into your clip.
A
GAME CLIP / HIGHLIGHTS
File: gse-thumb-type-a.html · Fades to black at 3s
16:9 Desktop
9:16 Shorts
✎ CHANGE THESE
◆
Main title — line 1 INSANE
The big top word. Use the emotion or action — CLUTCH, INSANE, CLEAN, FINAL, WILD. One word, all caps, as punchy as possible.
◆
Main title — line 2 MOMENT
The descriptor word. PLAY, WIN, CLIP, KILL, MOMENT, ROUND. Keep it one word.
◆
Category pill (top left) GAME CLIP
Swap for the game name — MARIO, WARZONE, ELDEN RING, FIFA. Max 2 words.
◆
Live indicator (top right) LIVE CAPTURE
Can stay as LIVE CAPTURE or change to the game title if you prefer.
🔒 NEVER CHANGE THESE
◆
Logo bug GREENS SCREENS ENT
Bottom-right corner. Always present. greensscreensent.com beneath it.
◆
GREENS.SCREENS sub-line GREENS.SCREENS
Appears beneath line 2 in accent lime. This is the brand handle. Do not remove.
◆
Diagonal slash animation
The scrolling background. The glitch bars. The corner marks. The fade-to-black timing. These are fixed design elements.
◆
Colors
#00FF88 green. #AAFF44 lime. #050A07 background. Never deviate.

03 · Type B
TYPE B — GAME REVIEWS
Use for: dedicated game reviews, first impressions, deep-dives, "is it worth it" videos, opinion pieces on a specific title. The score block is the hero — it signals immediately that this is an authoritative verdict, not just a clip.
B
GAME REVIEW / VERDICT
File: gse-thumb-type-b.html · Persistent (no fade)
16:9 Desktop
9:16 Shorts
✎ CHANGE THESE
◆
Game title — line 1 (green) GAME
First word or franchise name of the game. Examples: ELDEN, MARIO, HALO, ZELDA, FIFA.
◆
Game title — line 2 (white) TITLE HERE
Second word or subtitle. Examples: RING, KART 8, INFINITE, TOTK, 25. Keep it short enough to fit the frame.
◆
Score number 9.1
Your verdict score out of 10. One decimal place. This is the most visible element — make sure it reflects your actual take.
◆
Year in meta line 2026
Update to the year of the review if different from 2026.
🔒 NEVER CHANGE THESE
◆
Logo bug GREENS SCREENS ENT
Bottom-right. Always present.
◆
Meta line prefix GREENS.SCREENS REVIEW ·
The attribution text before the year. This stays. Only the year changes.
◆
REVIEW pill (top left) REVIEW
Always reads REVIEW on this template. Do not change to the game name.
◆
Score label / 10 · SIGNAL
The denominator and label beneath the score. Do not change.
◆
REVIEWED watermark, scan drift, side bar
All background animation elements are fixed. No stars — the numeric score is the only verdict.

04 · Type C
TYPE C — BRAND CONTENT
Use for: channel trailers, community updates, collab announcements, "about GSE" videos, newsletter reveals, brand milestones, or any video that isn't tied to a specific game. The radar and concentric rings signal broadcast — GSE reaching outward.
C
BRAND / GENERAL CONTENT
File: gse-thumb-type-c.html · Radar animation · Persistent
16:9 Desktop
9:16 Shorts
✎ CHANGE THESE
◆
Tagline (top center) TECHNOLOGY · GAMING · ENTERTAINMENT
Can stay as-is for most videos. For specific content, change to the topic — "COMMUNITY UPDATE", "NEW PARTNER · JOIN US", "SEASON 2 · INCOMING".
◆
ENT sub-line (accent lime) ENT
For most videos, leave as ENT. For milestone or special content only, this can become a short specific message — "SEASON 2", "NEW ERA", "100K". One word or phrase maximum.
◆
Pillar pills (bottom row) TECH · GAMING · ENT · COMMUNITY
These can be reduced or swapped for specific content focus. For a community update, you might show only COMMUNITY. For a tech video, only TECH. Default is all four.
🔒 NEVER CHANGE THESE
◆
GREENS / SCREENS wordmark GREENS · SCREENS
The two main lines are always GREENS (green) and SCREENS (white). These never change.
◆
SIGNAL ACTIVE indicator SIGNAL ACTIVE
Top right. Always present, always blinking. This is the brand heartbeat.
◆
Radar system
The sweep, pulse rings, crosshairs, blips. The full radar is a locked visual. It is what makes Type C unmistakably GSE.
◆
Background, grid, scanlines, bars
Deep forest dark background with radial green glow. The grid, top/bottom bars, and scanlines are all locked.

06 · Exporting
HOW TO EXPORT
Two ways to get your thumbnail out of the HTML file and into YouTube, TikTok, or wherever you need it.
AS AN MP4 — ANIMATED
1. Open OBS Studio
2. Add a Browser Source
3. Check "Local File" and select the HTML file
4. Set width/height to match the format:
   Desktop: 1920 × 1080
   Mobile: 1080 × 1920
5. Hit Record — capture the first 3–4 seconds
6. Use the mp4 as your video intro or animated thumb
AS A STILL IMAGE
1. Open the HTML file in Chrome
2. Wait for animations to settle (about 2 seconds)
3. Right-click → "Save as image" using a browser extension
   OR use the screenshot tool (Win: Snipping Tool / Mac: Cmd+Shift+4)
4. Crop to the exact canvas size if needed
5. Upload as your static YouTube or TikTok thumbnail
