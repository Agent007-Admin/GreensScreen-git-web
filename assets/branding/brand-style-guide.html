03 · Identity System
THE LOGO
Three approved logo configurations. Each serves a specific context. Never mix elements between configurations. Always use the provided SVG files — never reconstruct the logo by hand.
01 · Primary stacked
02 · Horizontal lockup
03 · Square icon / favicon
LOGO CLEAR SPACE RULE
Always maintain a minimum clear space around the logo equal to the height of the letter "G" in the wordmark. Nothing — text, imagery, or other graphic elements — should enter this protected zone.
MINIMUM SIZE
Primary stacked: 120px wide minimum
Horizontal: 200px wide minimum
Square icon: 32px minimum (24px favicon)
Logo usage rules
DO
◆Use on dark (#050A07) or black backgrounds
◆Maintain the clear space rule at all times
◆Use the provided SVG files exclusively
◆Keep ENT tag and signal dot together
◆Use square icon for Discord, favicons, app icons
DON'T
✕Stretch, skew, or alter proportions
✕Place on busy or uncontrolled backgrounds
✕Swap Bebas Neue for any other typeface
✕Use off-brand colors on the wordmark
✕Remove the ENT designation or signal dot
USE WHEN
→Primary stacked → merch, thumbnails, posters, alerts
→Horizontal → banners, headers, overlays, email
→Square icon → avatars, Discord, GitHub, favicon
→Monochrome → print, embroidery, merch tags
→Accent lime → secondary callouts and CTAs only

COLOR HIERARCHY
SIGNAL STRENGTH
Think of the palette as signal strength. #00FF88 is full broadcast — use it for things that demand attention. #AAFF44 is the secondary channel — accents and CTAs. #6B9A7A is ambient — supporting info that doesn't compete. Background colors are the silence the signal speaks into.
ACCESSIBILITY NOTE
DARK FIRST
The GSE palette is designed for dark environments. All primary content should live on #050A07 or #0F1E16. Never place primary green on a white background without using the approved monochrome variant. The brand is a screen brand — design for the screen first, print second.

06 · Voice
VOICE & TONE
How GSE speaks is as important as what it looks like. The brand voice is direct, sharp, and community-first. It doesn't over-explain. It doesn't undersell. It speaks like someone who knows exactly what they're building — and who they're building it for.
WE ARE
◆
Direct. We say what we mean. No fluff, no filler. Clear signal, clean transmission.
◆
Community-first. The audience isn't a number. They're partners in the signal. We speak with them, not at them.
◆
Tech-fluent. We know the tools. We reference the culture. We don't explain what a Discord server is — we build one worth joining.
◆
Ambitious. We talk about what we're building as if it's already real — because it is, in progress. Future-facing, not vague.
◆
Human. Behind the tech and the design is real personality. We're gamers. We're creators. We're builders. That shows up in the words.
WE ARE NOT
✕
Corporate. No brand speak. No "synergy." No "leveraging our ecosystem." Plain language, always.
✕
Arrogant. Confidence without community is just noise. We don't talk down. We invite in.
✕
Vague. "Coming soon" without context. "Big things ahead" without substance. If we speak, we say something real.
✕
Performative. We don't chase trends for attention. We create what's authentic to this brand and this community.
✕
Passive. Every sentence should feel like it has intent behind it. We don't post — we transmit.
WRITE LIKE THIS

✓ "Where the signal is always green."
✓ "Gaming evolved into a language. We speak it fluently."
✓ "The community isn't the audience — the community IS the brand."
✓ "Built different. Powered by the collective."

✕ "Stay tuned for exciting upcoming content!"

08 · Visual Language
THE AESTHETIC
The GSE visual language is retro-futurist tech — the feeling of a terminal that hasn't gone dark, a signal that never drops, a screen that's always on. Dark, electric, precise.
TEXTURE
SCANLINES
Subtle horizontal scanlines overlay all primary surfaces. This is not decoration — it is atmosphere. It references the CRT screen, the broadcast monitor, the signal in transmission. Keep it subtle: 2–4px repeating at 3–4% opacity.
STRUCTURE
GRID BG
A fine green-tinted grid lives behind all page content. 48×48px cells at 3% opacity on the primary green. This references the coordinate system, the map grid, the digital terrain. It is always present, always subtle.
SHAPE LANGUAGE
CLIP PATH
Buttons and interactive elements use the GSE clip-path: a parallelogram with angled left and right edges. This creates a sense of forward motion and precision. All CTAs, badges, and active UI elements use this shape.
MOTION
SIGNAL DOT
The pulsing signal dot is the brand's heartbeat. It appears in the logo, on live indicators, and on active states. A 2-second ease-in-out pulse between 100% and 120% scale with a glow. It signals presence, activity, and broadcast.
BORDER
GREEN LINE
Active cards reveal a 2px green line at the top that scales from 0 to 100% on hover. This is the signature interaction. It appears on pillar cards, social cards, portfolio items. It means: this element is live. This element has signal.
CURSOR
THE BLINK_
The blinking cursor (a square block, not a line) appears after key brand phrases: "THE UMBRELLA_", "SIGNAL_ALWAYS_GREEN", "GREENS.SCREENS_". It implies the brand is live, typing, in progress. It is never decorative — always intentional.
